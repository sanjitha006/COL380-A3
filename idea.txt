Rank 0:
  1. Read input, broadcast graph to all ranks (MPI_Bcast)
  2. Expand root to depth 1/2, collect subproblems
  3. Distribute one subproblem per rank (round-robin)
  4. Store overflow in master queue
  5. Enter coordination loop:
       - Listen for: WORK_REQUEST | NEW_BEST | DONE
       - On WORK_REQUEST: send next subproblem or EMPTY signal
       - On NEW_BEST: MPI_Ibcast updated P_max to all
       - On DONE: decrement active counter
       - Exit when active == 0 AND master queue empty
  6. MPI_Reduce to collect final best, print output

Ranks 1…P−1:
  1. Receive graph (MPI_Bcast)
  2. Receive initial subproblem from rank 0
  3. Loop:
       a. MPI_Iprobe for incoming STEAL_REQUEST → if queue large, split+send
       b. MPI_Test on pending P_max broadcast → update local P_max
       c. Pop subproblem from local queue
       d. Run colorBound → prune or continue
       e. Run knapsackBound → prune or continue
       f. Branch: push children to local queue
       g. If new best found → MPI_Isend(NEW_BEST, rank 0)
       h. If local queue empty → MPI_Send(WORK_REQUEST, rank 0)
            → if EMPTY response AND no steal succeeds → send DONE, exit
